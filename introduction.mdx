## Overview

Snacko empowers you to build smarter eating habits with personalized nutrition plans and seamless tracking. Powered by behavior science, the platform helps individuals and dietitians make healthier choices effortlessly. Track your intake, receive tailored recommendations, and achieve your wellness goals without the hassle.

<Callout kind="info">
Snacko combines nutrition tracking with psychological insights to create sustainable habit changes.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at https://dashboard.snacko.com/signup.
  </Step>
  <Step title="Set Goals" icon="target">
    Answer a quick quiz to generate your personalized nutrition plan.
  </Step>
  <Step title="Track First Meal" icon="utensils">
    Log your breakfast using the mobile app or web dashboard.
  </Step>
  <Step title="Review Insights" icon="bar-chart-3">
    Check your daily summary and behavior tips.
  </Step>
</Steps>

For developers integrating Snacko, start with the API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/plans', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const plans = await response.json();
  console.log(plans);
  ```
  ```python
  import requests
  response = requests.get(
    'https://api.example.com/v1/plans',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  plans = response.json()
  print(plans)
  ```
</CodeGroup>
